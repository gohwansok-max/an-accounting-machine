✅ 완성

🔧 필수 설정

1

Google Gemini API 키 발급

AI가 문제를 자동 생성하기 위한 필수 단계입니다.

⚠️ 2024년 6월 기준
Google의 Gemini API는 무료 사용 가능합니다. 비용 청구가 되지 않습니다. (일일 한도 60회 요청 무료)
2

앱 설정에 키 입력

발급받은 API 키를 앱에 입력합니다.

단계:
1️⃣ index.html 열기
2️⃣ ⚙️ 설정 버튼 클릭
3️⃣ "Google Gemini API 키" 입력란에 키 붙여넣기
4️⃣ "💾 저장하기" 클릭
✅ 완료!
3

첫 문제 생성

이제 AI가 맞춤형 문제를 자동 생성할 수 있습니다!

1️⃣ 탭 선택 (예: "📮 우편일반")
2️⃣ 난이도 선택 (예: "실전")
3️⃣ "🔄 새 문제" 클릭
✨ AI가 20개 문제 자동 생성!

✨ 주요 기능

  • ✓ AI 맞춤형 문제 생성 완성

    Gemini 3.5 Flash/Pro로 2024년 이후 최신 데이터 기반 문제 자동 생성

  • ✓ 4개 과목별 특화 문제 완성

    우편일반, 예금일반, 보험일반, 컴퓨터일반 각 과목 맞춤형 출제

  • ✓ 자료 기반 문제 생성 완성

    PDF/URL/텍스트 자료를 입력하면 자동 분석해서 문제 생성

  • ✓ 공식 자료 검색 선택

    Custom Search API로 우정사업본부 공식 자료 실시간 검색 (선택 설정)

  • ✓ 자동 오답 분석 완성

    틀린 문제 자동 기록 및 AI 약점 분석

  • ✓ 게이미피케이션 완성

    레벨/XP/뱃지로 학습 동기 부여

  • ✓ 딥블루 프로페셔널 UI 완성

    공무원 수험앱 특화 디자인, 모바일 최적화

❓ 자주 묻는 질문

Q. API 키 없이도 사용 가능한가요?

네! 기본 문제(12개)로는 키 없이 시작 가능합니다. 하지만 AI 맞춤형 문제 생성을 위해서는 Gemini API 키가 필수입니다.

Q. 비용이 드나요?

모두 무료입니다. Gemini API와 Custom Search API 모두 무료 할당량이 충분합니다. (2024년 6월 기준)

Q. 오프라인에서도 사용 가능한가요?

기본 문제와 사전 생성된 AI 문제는 오프라인에서도 사용 가능합니다. 새로운 AI 생성은 인터넷이 필수입니다.

Q. 여러 기기에서 동기화되나요?

현재는 각 기기별 독립적입니다. 향후 업데이트에서 클라우드 동기화 추가 예정입니다.
