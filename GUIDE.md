## 🚀 5분 빠른 시작

### Step 1: Google Gemini API 키 발급 (2분)

```
✅ 단계별 가이드:

1️⃣ https://ai.google.dev/gemini-api 접속
   → Google 계정으로 로그인 (Gmail, Google 워크스페이스 등)

2️⃣ "Get started" 또는 "Create API Key" 클릭
   
3️⃣ "API 키 생성" 버튼 클릭
   - 새 프로젝트 또는 기존 프로젝트 선택 가능
   - "Get API Key" 최종 클릭
   
4️⃣ 생성된 긴 문자열 (AIza...) 복사

5️⃣ 계리패스 AI 앱 접속
   → ⚙️ 설정 클릭
   → "Google Gemini API 키" 입력란에 붙여넣기
   → "💾 저장하기" 클릭

✅ 완료! 이제 AI 문제 생성 가능
```

### Step 2: 첫 문제 풀어보기 (2분)

```
1. 탭 선택
   예: "📮 우편일반" 클릭
   
2. 난이도 선택
   예: "실전" 선택 (처음이면 "기본" 권장)
   
3. 문제 로드
   → "🔄 새 문제" 클릭
   → 또는 자동으로 기본 문제 표시

4. 풀어보기
   → 4지선다 선택 (클릭)
   → 또는 주관식 입력

5. 채점
   → "✅ 채점하기" 클릭
   → 점수 및 해설 확인
   → 틀린 문제는 자동 오답노트 저장
```

### Step 3: 오답 노트 확인 (1분)

```
✅ 채점 직후 자동 표시되는 "오답 노트" 섹션:
- 틀린 문제 번호
- 정답 및 해설
- 다운로드 버튼 클릭하면 텍스트 파일로 저장
```

## 🔧 상세 설정 가이드

### 필수: Google Gemini API

#### 발급 절차 상세

**상황 1: Google 계정이 있는 경우**
```
1. https://ai.google.dev 접속
2. 우측 상단 "Get API Key" 버튼 클릭
3. 팝업에서 "Create API Key" 선택
4. 기존 Google Cloud 프로젝트 선택 또는 새로 생성
5. 생성된 키 자동 복사
6. 앱의 ⚙️ 설정에 붙여넣기
```

**상황 2: Google Cloud 프로젝트가 있는 경우**
```
1. https://console.cloud.google.com 접속
2. 프로젝트 선택 또는 생성
3. "APIs & Services" → "Credentials"
4. "Create Credentials" → "API Key"
5. 생성된 키 복사
6. 앱 설정에 입력
```

**상황 3: 키 생성이 안 되는 경우**
```
⚠️ 확인 사항:
□ Gmail 계정으로 로그인했나?
□ 신용카드 정보 등록했나? (비용 청구 안 됨 — 무료 할당량 충분)
□ 이중 인증(2FA) 활성화했나?
□ 다른 브라우저로 시도해보기
```

#### API 키 안전성

```
🔐 보안 팁:
✅ 이 기기(localStorage)에만 저장
❌ 메일/메시지로 공유 금지
❌ 공개 저장소에 커밋 금지
⚠️ 의심 활동 시 Console에서 바로 삭제 가능
```

### 선택: Google Custom Search API (공식 자료 검색)

#### 설정 절차

```
🔍 공식 자료 검색 기능을 원하면:

1단계: Google Cloud Console에서 API 활성화
  └─ https://console.cloud.google.com
     → 프로젝트 선택
     → "APIs & Services" 검색
     → "Custom Search API" 찾기
     → "Enable" 클릭

2단계: API 키 생성
  └─ "Credentials" 탭
     → "Create Credentials" → "API Key"
     → 키 복사 (skey로 저장)

3단계: Custom Search Engine(cx) 생성
  └─ https://programmablesearchengine.google.com
     → 로그인
     → "Create" 클릭
     → "Search the entire web" 선택
     → "Create" 클릭
     → 생성된 cx ID 복사 (cxid로 저장)

4단계: 앱 설정
  └─ ⚙️ 설정 → Google Custom Search API 섹션
     → "Google Custom Search API 키" 입력 (skey)
     → "Google Search Engine ID" 입력 (cxid)
     → "💾 저장하기"

✅ 완료! 이제 📄 자료 입력에서 "검색" 기능 활성화됨
```

## ❓ FAQ

### Q1: 생성된 문제가 이전 것과 같아요
**A:** 
```
✅ 해결책:
1. "🔄 새 문제" 클릭 후 약 2-3초 기다리기
2. 인터넷 연결 확인 (와이파이)
3. 모델 변경해보기 (⚙️ 설정 → gemini-1.5-flash로)
```

### Q2: API 키는 안전한가요?
**A:**
```
✅ 100% 안전합니다:
- 로컬 스토리지(이 기기)에만 저장
- 서버로 전송되지 않음
- 브라우저 닫아도 유지됨 (필요시 수동 삭제)
- "🗑 모두 지우기" 클릭하면 즉시 삭제

⚠️ 주의:
- 다른 사람이 사용할 컴퓨터에서는 "모두 지우기" 후 종료
- 공개 와이파이 사용 시 VPN 권장
```

### Q3: 오답노트는 어디에 저장되나요?
**A:**
```
📍 저장 위치:
1. 로컬 스토리지 (이 기기)
   └─ 브라우저 데이터 삭제 시 함께 삭제될 수 있음

2. 다운로드 폴더 (📥 다운로드 클릭 시)
   └─ .txt 파일로 저장되어 영구 보관 가능

💡 팁: 주 1회는 다운로드해서 백업하세요!
```

### Q4: 인터넷이 없으면 사용 가능한가요?
**A:**
```
⚠️ 제한적 사용 가능:

🔴 불가능한 것:
- AI 문제 생성 (API 필요)
- 자료 분석 (API 필요)
- 공식 자료 검색

🟢 가능한 것:
- 기본 문제 풀이 (사전 로드된 문제)
- 채점 및 오답노트
- 학습 통계 확인

💡 팁: 미리 "🔄 새 문제"로 문제들을 생성해두면
     오프라인에서도 학습 가능합니다!
```

### Q5: 여러 기기에서 진행상황을 동기화할 수 있나요?
**A:**
```
❌ 자동 동기화 없음 (현재)

🔄 대안:
1. 한 기기에서만 주로 사용
2. 다른 기기 사용 시:
   ├─ 브라우저 프로필/계정 동기화 활용
   ├─ 또는 매번 설정 다시 입력
3. 오답노트 정기 다운로드 → 클라우드 저장

💡 향후 업데이트에서 클라우드 동기화 예정
```

├─ Step 4: 키 입력 후 "💾 저장하기" 클릭
└─ Step 5: "🔄 새 문제" 다시 시도

□ 키가 "AIza"로 시작하나?

### 문제 2: "HTTP 403" 또는 "인증 실패" 오류

```
원인: API 키 무효 또는 한도 초과

해결책:
┌─ 새 키 발급 시도
│  └─ https://ai.google.dev/gemini-api
│  └─ "Create new API Key" 클릭
│  └─ 새 키로 교체
├─ 한도 확인
│  └─ https://console.cloud.google.com
│  └─ Quotas & System Limits 확인
└─ 재시도

💡 팁: 무료 할당량이 충분하면
  보통 24시간 후 복구됩니다.
```

### 문제 3: 문제가 생성되지만 형식이 이상해요

```
원인: JSON 파싱 오류 또는 API 응답 문제

해결책:
┌─ 브라우저 콘솔 확인
│  └─ F12 → Console 탭
│  └─ 빨간 오류 메시지 확인
├─ 모델 변경
│  └─ ⚙️ 설정 → "gemini-1.5-flash" 선택
├─ 인터넷 재연결
└─ 다른 브라우저 시도

💡 콘솔 오류를 캡처해서
  개발진에게 보고하면 더 빨리 해결됩니다.
```

### 문제 4: 오답노트가 저장되지 않음

```
원인: 로컬 스토리지 비활성화 또는 용량 부족

해결책:
┌─ 브라우저 설정 확인
│  └─ Chrome: 설정 → 개인정보 → 쿠키 및 기타 사이트 데이터
│  └─ Firefox: 설정 → 개인정보 → 히스토리
│  └─ Safari: 환경 설정 → 개인정보
├─ 시크릿/프라이빗 모드 종료
│  └─ 일반 모드에서 사용하기
├─ 브라우저 캐시 삭제
│  └─ Ctrl+Shift+Delete (Windows)
│  └─ Cmd+Shift+Delete (Mac)
└─ 저장 공간 확인
   └─ PC 저장공간 20GB 이상 확인

✅ 해결 후 채점 다시 시도
```
