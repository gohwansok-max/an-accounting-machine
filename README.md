### 1. **AI 맞춤형 문제 생성**

- Gemini AI의 개인화 피드백 및 학습 방향 제시
- 오답 노트 다운로드 및 통계 추적

### 4. **게이미피케이션**
- 레벨/경험치(XP) 시스템
- 과목별 마스터 뱃지
- 별점 누적 시스템

### 5. **프로페셔널 UI/UX**
- 공무원 수험앱 특화 딥블루 테마
- 모바일 최적화
- 인쇄 기능

## 🚀 빠른 시작

### 필수 설정

#### 1. Google Gemini API 키 발급
```
1. https://ai.google.dev/gemini-api 접속
2. "Get API Key" 클릭
3. Google Cloud 프로젝트 생성
4. API 키 복사
5. 앱의 ⚙️ 설정 → "Google Gemini API 키" 입력
```

#### 2. Google Custom Search API 설정 (선택)
```
1. https://console.cloud.google.com 접속
2. "Custom Search API" 활성화
3. API 키 및 Search Engine ID 복사
4. 앱 설정에 입력 → 공식 자료 검색 활성화
```

#### 3. OpenAI API 키 (선택 — 음성 기능)
```
1. https://platform.openai.com/account/api-keys 접속
2. API 키 생성
3. 앱 설정에 입력
```

## 🔧 API 설정 상세 가이드

### Google Gemini API 키 (필수)

**발급 방법:**
```bash
1. https://ai.google.dev 접속
2. "Get API Key" → "Create API Key"
3. 새 프로젝트 생성 또는 기존 프로젝트 선택
4. 자동 생성된 API 키 복사
```

**모델 선택 (⚙️ 설정에서):**
- `gemini-2.0-flash` (빠름, 추천)
- `gemini-1.5-pro` (정확함)
- `gemini-1.5-flash`

### Google Custom Search API (선택)

**설정 순서:**
```
1. https://console.cloud.google.com 접속
2. 프로젝트 생성
3. "Custom Search API" 검색 → 활성화
4. 사용자 인증 정보 → API 키 생성 (타입: API 키)
5. Custom Search Engine 생성:
   https://programmablesearchengine.google.com
   - "create" → 검색 사이트 설정
   - "cx" (Search Engine ID) 복사
6. 앱 설정에 API 키와 cx ID 입력
```

### OpenAI API 키 (선택)

**음성 읽어주기 기능 활성화:**
```
1. https://platform.openai.com/account/api-keys
2. "Create new secret key"
3. 앱 설정에 입력
4. 지문 읽어주기 버튼 활성화
```

## 🎓 학습 전략

### 📚 추천 학습 순서

1. **기본 단계 (1~2주)**
   - 난이도: "기본"
   - 목표: 과목별 핵심 개념 이해
   - 방법: 문제풀이 후 해설 정독

2. **실전 단계 (2~3주)**
   - 난이도: "실전"
   - 목표: 실제 시험 수준 적응
   - 방법: 오답노트 정리 + 개념 복습

3. **심화 단계 (1~2주)**
   - 난이도: "심화"
   - 목표: 최종 점검 및 취약점 보완
   - 방법: 오답노트 집중 학습

### 💡 효과적 활용 팁

- **일일 목표 설정**: 하루에 50~60문제 풀기
- **오답노트 정리**: 매주 1회 오답 분석 시간 확보
- **자료 활용**: 최신 법령/정책 자료 주 2회 입력
- **레벨업 추적**: XP 누적으로 학습 진도 확인

## 🐛 문제 해결

### Q: 문제가 생성되지 않음
**A:** 
- ⚙️ 설정에서 Gemini API 키 입력 확인
- 인터넷 연결 확인
- 브라우저 콘솔(F12)에서 오류 메시지 확인

### Q: 자료 분석이 느림
**A:**
- 첫 실행은 시간이 걸릴 수 있습니다
- 자료 텍스트 크기를 3000자 이내로 유지
- 모델을 "gemini-2.0-flash"로 변경해보세요

### Q: 오답노트가 저장되지 않음
**A:**
- 브라우저 로컬 스토리지 활성화 확인
- 시크릿 모드에서는 데이터가 저장되지 않습니다
- 정상 모드에서 접속하세요

### Q: 음성 기능이 작동하지 않음
**A:**
- OpenAI API 키 입력 필수
- 또는 브라우저 기본 음성 사용 가능
- Chrome, Safari 권장
